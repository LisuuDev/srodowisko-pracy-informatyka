# ✨ Inspiracje i Podstawy Projektu

## Źródła Pomysłu
* **Własna Potrzeba:** Uczucie frustracji i chaosu podczas sesji egzaminacyjnych.
* **Własna Analiza:** Konieczność połączenia funkcji list zadań (To-Do) z wizualizacją kalendarzową.
* **Własne Obserwacje:** Analiza popularnych aplikacji typu Planner/Trello/Google Calendar pod kątem ich zalet i wad dla specyficznej grupy studentów.
